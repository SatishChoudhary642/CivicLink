# **App Name**: CivicLink

## Core Features:

- Issue Reporting: Users can report civic issues with title, description, category, image, and location. Image can be directly clicked from camera on app and will be uploaded with geo tag
- Interactive Map Display: Display reported issues as pins on an interactive map with status-based color-coding.
- Voting System: Registered users can upvote or downvote reports to confirm or flag them.
- Status Prediction: AI predicts the resolution status of a report based on its category, description and location using a trained machine learning tool.
- Admin Dashboard: Secure dashboard for admins to manage reports and update statuses.
- Report Filtering: Admins can filter reports by status and category for efficient management.
- Automated Notifications: System sends push notifications to reporters when the status of their report changes.
- Image Categorization: Uploaded image is categorized by AI to improve report accuracy and routing.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to convey trust and civic responsibility.
- Background color: Light gray (#F0F4F8) to provide a clean and neutral backdrop.
- Accent color: Vibrant orange (#FFB347) for call-to-action buttons and important notifications.
- Body font: 'PT Sans', a humanist sans-serif, will be used for main text elements to give a clean modern look. Headline font: 'Playfair', a modern sans-serif, for headlines.
- Use clear and recognizable icons from a set like Material Design Icons to represent report categories.
- Implement a responsive design that adapts to different screen sizes, ensuring usability on both desktop and mobile devices.
- Use subtle transitions and animations to provide feedback to user interactions, such as button presses and status updates.
